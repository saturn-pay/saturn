# Saturn Dashboard

Next.js admin dashboard for monitoring Saturn agents, wallets, transactions, and audit logs. Built with Tailwind CSS.

## Running

```bash
cd web
npm install
npm run dev
```

The dashboard runs on [http://localhost:3000](http://localhost:3000). The main Saturn API server must be running for the dashboard to work.
