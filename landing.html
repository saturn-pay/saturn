FAQ
Common questions
Why Bitcoin / Lightning instead of USD?

Lightning is a machine-native payment rail: global, instant, programmable, and granular enough for sub-cent per-call pricing. No chargebacks, no bank hours, no currency conversion for international agents. It’s an infrastructure choice — fiat on-ramps can be added later.

What APIs are in the service catalog?

Typical catalog: LLMs (OpenAI, Anthropic), search, email, and code execution. Add services over time and request integrations.

Can I use Saturn with any agent framework?

Yes. The Saturn SDK is a thin client. It works with LangChain, CrewAI, AutoGen, custom loops — anything that can make an API call.

What happens if an agent exceeds its budget?

The call is rejected before it reaches the upstream provider. Your agent gets a clear policy error with remaining sats. No overspend.

How does Saturn handle API keys?

Saturn manages upstream API credentials. Your agents use agent-scoped tokens to authenticate to Saturn — they never see provider keys.
